## C语言实现大小写的转换

这类题目主要通过ASCII（美国信息交换标准）代码码差值实现，A对应ASCII码十进制数字是65，a对应ASCII码十进制数字是97，即大小写字母之间ASCII码差值为32，想要将大写字母转换为小写字母可以将该字符ASCII码值+32，同理小写字母转换成大写字母只需将该字符ASCII码值-32

使用代码如下：

```c
#include <stdio.h>

int main()
{
	char ch;
	scanf("%c", &ch);
	ch = ch - 32;
	printf("%c",ch);
	return 0;
}
```
